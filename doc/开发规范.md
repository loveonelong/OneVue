# Vue团队开发规范

> 目前本规范主要针对vue-cli构建的项目

## 目录结构约定

```
-为目录 *为文件

- src
  - assets ------------------------ 静态资源都放在这里
  	- img ------------------------- 图片
  	- style ----------------------- 样式
  - common ------------------------ 所有构成公用模块的模块（原子模块）
  	* A.vue ----------------------- B和C模块公用的A模块
  - components -------------------- 所有的公用模块
  	* B.vue ----------------------- B公用模块
  	* C.vue ----------------------- C公用模块
  - router ------------------------ 路由
  	* routes.js ------------------- 
  	* index.js -------------------- 
  - views ------------------------- 所有的页面，及其页面内部的私有模块，该目录下所有的目录、组件名称均采用首字母大写，按照url关系存放
    - Home ------------------------ home页面的目录
      * Home.vue ------------------ home页面的主模块
      * Banner.vue ---------------- home页面中的私有模块
    - Goods ----------------------- 商品页面的目录
      * Goods.vue ----------------- 商品页面的主模块
        - Goods
        - GoodsDetails ------------ 商品详情的目录
          * GoodsDetails.vue ------ 商品详情的主模块
          * GoodsDetailsList.vue -- 商品详情的某个私有模块
  - api --------------------------- 这里存放所有跟请求有关的文件
    * api.js ----------------------- 请求文件
    
```
